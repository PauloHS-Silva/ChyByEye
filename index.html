Welcome to ChiByEye

ChiByEye is an interactive GUI for visualizing and manipulating XSPEC astrophysical models. Built using PyQt5 and Matplotlib, it allows you to explore how model parameters affect the overall spectral shape.

About

This software enables intuitive exploration of XSPEC models. You can load models manually or from XCM files, adjust parameters, visualize spectral components, and perform basic fits. Ideal for educational and research purposes.

Read the full documentation (PDF)

Download

To run the software, you need:

  • Python 3
  • XSPEC and pyxspec
  • numpy, matplotlib, and PyQt5

You can download the latest version from our GitHub repository.
